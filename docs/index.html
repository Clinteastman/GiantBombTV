Features

  • Browse recent videos, shows, and categories
  • Continue Watching with server-synced progress and resume
  • Watchlist management
  • Search across all Giant Bomb videos
  • Stream quality selection: Auto, 1080p, 720p, 480p, 360p
  • Chromecast support for casting to your TV
  • Picture-in-Picture mode on phones
  • Upcoming shows schedule and live streams
  • Show browsing with infinite scroll through episodes

Getting Started

  1. Download the latest release from GitHub
  2. Install the APK on your device
  3. Log in to giantbomb.com and find your API key in your profile settings
  4. Launch the app and enter your API key

A free Giant Bomb account gets you access to some content. Giant Bomb Premium unlocks exclusive shows, ad-free videos, and the full back catalog.
